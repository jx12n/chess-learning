# Chess — Day 7: Piece Values & The Whole Army

**For:** a 7-year-old beginner · **Time:** ~20 min, in short bursts · **You need:** a real board + pieces (or just this sheet)

**Today's rule of the day:** *When you can only grab one prize, grab the biggest one.* Today has no new moves at all — it's the week's victory lap: every piece they've learned, working together, making smart choices.

**Goal for today:** by the end, your child can say what each piece is worth, pick the more valuable capture when offered two, and solve a puzzle using two different pieces as a team.

---

## Part 1 — The prize chart (4 min) 🏆

Every piece has a value — think of it as points, or cookies:

```
  pawn ..... 1        "little soldier"
  knight ... 3        "the jumper"
  bishop ... 3        "the slider"
  rook ..... 5        "the tower"
  queen .... 9        "the superstar"
  king ..... ∞        "the whole game — never for sale"
```

**Quick game — "Which is bigger?":** rapid-fire pairs. "Rook or bishop?" (rook, 5 beats 3) "Knight or pawn?" (knight) "Two rooks or a queen?" (tricky — 10 beats 9, two rooks!). Keep it bouncy; the numbers stick from play, not drilling.

---

## Part 2 — One move, one prize (6 min)

The big skill: **look at the WHOLE board before you grab.** Your rook can take the knight (3) up the file, or the pawn (1) along the row — but you only get ONE move. Which one?

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   | n |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   | R |   |   |   | p |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

(The knight — 3 beats 1.) Run a few more from the app or set your own: bishop choosing rook-vs-pawn, queen choosing queen-vs-rook, knight choosing rook-vs-pawn. Same question every time: *"Which prize is bigger?"*

**Coaching, gently:** if they grab the little one, don't say "wrong" — say *"you got 1 point… was there a 3 on the board?"* Let the numbers do the teaching.

---

## Part 3 — Mini-game: "The Whole Army" (8 min) ⚔️

The week's finale: TWO of your pieces, one mission, and a move budget so tight that **both pieces must pitch in.** Rook and bishop, four pawns, four moves — every move must gobble:

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 | p | p |   |   | p |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   | p |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 | R |   |   |   |   | B |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

Ask before they touch anything: **"Which pawns can the rook reach? Which can only the bishop reach?"** (The d3 pawn is on a diagonal from f1 — that one is bishop food. The rook hoovers the rank-5 crowd: a5, then b5, then e5.) Planning WHO takes WHAT is the whole game — that's teamwork, and it's how real chess feels.

Two more army puzzles wait in the app: rook + knight, then queen + knight. Same question every time: *who takes what?*

---

## Wrap-up — the week (3 min) 🎉

Have them teach YOU the whole week, one line per piece:

- **Rook** — plus sign, can't jump.
- **Bishop** — X, one color forever.
- **Queen** — both at once, the 9-point superstar.
- **Knight** — the L, the only jumper.
- **Pawn** — march straight, munch sideways, dreams of queening.
- **King** — one careful step; lose him, lose the game.

That's the entire alphabet of chess. They can *read the board* now.

**Legend:** `R`/`B` = your rook/bishop · `n`/`p` = enemy knight/pawn · `★` = the target square

**Stop while it's still fun.** Next adventure, when you're both ready: **check and checkmate** — how games are actually won. The app will unlock it as the lessons are built; the pieces they've mastered this week are everything those lessons stand on.
