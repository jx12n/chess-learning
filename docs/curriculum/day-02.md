# Chess — Day 2: The Bishop

**For:** a 7-year-old beginner · **Time:** ~20 min, in short bursts · **You need:** a real board + pieces (or just this sheet)

**Today's rule of the day:** *Bishops only ever slide diagonally — and each bishop stays on its own color forever.* If they remember one thing today, that's the thing.

**Goal for today:** by the end, your child can (1) slide a bishop along a diagonal without drifting off it, and (2) capture with it.

---

## Part 1 — The bishop makes an X (4 min)

Put one bishop (`B`) on **d4**. Ask first: "The rook made a plus sign. What shape do you think THIS one makes?" Then show them — the `*` squares are everywhere this bishop can go:

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   | * |
  +---+---+---+---+---+---+---+---+
7 | * |   |   |   |   |   | * |   |
  +---+---+---+---+---+---+---+---+
6 |   | * |   |   |   | * |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   | * |   | * |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   | B |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   | * |   | * |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   | * |   |   |   | * |   |   |
  +---+---+---+---+---+---+---+---+
1 | * |   |   |   |   |   | * |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

Say it like this: **"The bishop makes an X. Never straight."** Have them trace both diagonals with a finger.

**The magic secret:** look at every `*` square — they are ALL the same color as d4. A bishop lives on one color its whole life. Ask: "Can this bishop EVER reach a square of the other color?" (No! Let them try to find one.)

---

## Part 2 — Blocking & capturing, bishop edition (4 min)

Same two rules as the rook, new shape:
1. **Bishops can't jump.** A friend on the diagonal stops them early.
2. **To capture,** land exactly on the enemy's square.

Here `P` is a **friendly** pawn and `x` is an **enemy**. Going up-right, the bishop must stop at e5 (friend on f6 blocks it). Going up-left it can go to c5 or capture on **b6**, but never past it:

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   | x |   |   |   | P |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   | * |   | * |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   | B |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   | * |   | * |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   | * |   |   |   | * |   |   |
  +---+---+---+---+---+---+---+---+
1 | * |   |   |   |   |   | * |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

**Ask them:** "Can the bishop reach g7?" (No — the friend on f6 blocks the road.) "Can it eat the enemy on b6?" (Yes — land right on it.) "Can it slide on to a7 after that?" (Not this move — capturing ends the slide.)

## Part 3 — The pawn in the way (4 min)

Day 1's big lesson returns in diagonal form. Bishop on **f1** wants to reach **a6** — but your own pawn on d3 sits right on the road. Ask: "Straight there? No. So how?" Let them discover the detour: **f1→h3→c8→a6** (three slides, around the back).

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 | ★ |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   | P |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   | B |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

## Part 4 — Mini-game: "Bishop Gobble" (6 min) 🍪

The main event. One bishop, five pawns — all sitting on the bishop's color (they have to be, or he could never eat them!). The pawns don't move. Gobble **all five**.

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   | p |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   | p |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   | p |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   | p |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   | B |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   | p |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

*(One clean route: d4, c5, then slide to e7, f6, g7, and finish on b2 — six moves. Don't chase perfection; finishing is the win.)*

**Coaching, gently:** when they drift off the diagonal, don't say "wrong" — ask *"is that square the bishop's color?"* That one question fixes almost every bishop mistake.

---

## Wrap-up (2 min)

Have them teach it back: "How does a bishop move?" (An X. Only diagonals. Can't jump. Lives on one color forever.)

**Legend:** `B` = your bishop · `p` = enemy pawn · `P` = your pawn · `x` = enemy you can capture · `*` = a square the bishop can reach · `★` = the target square

**Stop while it's still fun.** Next time: the queen — a rook and a bishop wearing one crown. Tell them that much and watch the eyes go wide.
