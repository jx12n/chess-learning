# Chess ‚Äî Day 5: The Pawn

**For:** a 7-year-old beginner ¬∑ **Time:** ~20‚Äì25 min, in short bursts ¬∑ **You need:** a real board + pieces (or just this sheet)

**Today's rule of the day:** *Pawns march STRAIGHT but gobble DIAGONALLY.* It's the only piece that captures differently from how it moves ‚Äî that's the tricky bit, and it's also today's whole story. Plus the best rule in chess: reach the far end, become a QUEEN.

**Goal for today:** by the end, your child can (1) march a pawn (one step, or two from its start), (2) capture with the diagonal bite, and (3) promote one and shout about it.

---

## Part 1 ‚Äî Marching (4 min)

Pawns are little soldiers: they only walk **forward**, one square at a time ‚Äî never sideways, never backward. Special deal on the very first move: a pawn *may* take **two** steps.

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   | * |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   | * |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   | P |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

**Quick game:** "March to e4 in ONE move." (The double step!) Then: "Now march one more." (e5 ‚Äî singles only, from now on.)

---

## Part 2 ‚Äî The diagonal bite (6 min) ü¶∑

Here's the weird, wonderful part. A pawn **cannot** capture the piece right in front of it ‚Äî the two just stand there, nose to nose, stuck forever. It captures **one step diagonally forward**, like a tiny bishop bite.

Nose-to-nose (nobody can do anything ‚Äî try it!):

```
  +---+---+---+---+---+---+---+---+
4 |   |   |   | p |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   | P |   |   |   |   |
  +---+---+---+---+---+---+---+---+
```

The bite (P on e4 can eat EITHER enemy ‚Äî but can't step forward):

```
  +---+---+---+---+---+---+---+---+
5 |   |   |   | x | p | x |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   | P |   |   |   |
  +---+---+---+---+---+---+---+---+
```

**Ask them:** "Can e4 eat the pawn straight ahead on e5?" (Never!) "Where CAN it bite?" (d5 or f5.) Chant it together: **"march straight, munch sideways."**

Then the zig-zag snack run ‚Äî three pawns, eaten one bite at a time (e4 takes d5, then e6, then f7):

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   | p |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   | p |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   | p |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   | P |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

## Part 3 ‚Äî The pawn's dream: PROMOTION (6 min) üëë

Tell it like a story: every pawn dreams of crossing the whole board ‚Äî because a pawn that reaches the far end **transforms into a QUEEN** (a new one ‚Äî you can even have two queens!).

March **b5 ‚Üí b6 ‚Üí b7 ‚Üí b8** and swap the pawn for a queen. Make it theatrical. Gasp.

Then the grand version ‚Äî gobble UP THE STAIRCASE, and the last bite lands on the top row: **e5 takes f6, takes g7, takes the rook on h8 ‚Äî and CHOMP, promotion on a capture!**

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   | r |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   | p |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   | p |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   | P |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

## Part 4 ‚Äî Mini-game: "March, then munch" (5 min) üç™

Pawn on **c2**, three pawns on a staircase just out of reach. The lesson hiding inside: sometimes the small step beats the double step. One quiet march to c3 puts d4 in the bite ‚Äî then it's munch, munch, munch. **Four moves: one march, three gobbles.**

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   | p |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   | p |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   | p |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   | P |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

**Coaching, gently:** the classic mistake is reaching straight ahead to capture. Don't correct ‚Äî just ask *"do pawns munch straight or sideways?"* and let the chant do the work.

---

## Wrap-up (2 min)

Teach-back: "How do pawns move?" (Straight ahead ‚Äî one step, two on the first move.) "How do they capture?" (Diagonally, one step.) "And what's the dream?" (Reach the end, become a QUEEN!)

**Legend:** `P` = your pawn ¬∑ `p` = enemy pawn ¬∑ `r` = enemy rook ¬∑ `x` = enemy you can capture ¬∑ `*` = a square the pawn can reach ¬∑ `‚òÖ` = the target square

**Stop while it's still fun.** Next time: the king ‚Äî the most important piece on the board. Slow, careful, and the whole game depends on him.
