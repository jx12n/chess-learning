# Chess — Day 1: The Board & The Rook

**For:** a 7-year-old beginner · **Time:** ~30 min, in short bursts · **You need:** a real board + pieces (or just this sheet)

**Today's rule of the day:** *We do NOT learn checkmate yet.* Day 1 is about the board and moving one piece cleanly. That slow start is on purpose — kids who nail the basics first improve much faster later.

**Goal for today:** by the end, your child can (1) point to any square, and (2) move a rook correctly and capture with it.

---

## Part 1 — Meet the board (3 min)

Set the empty board in front of them. Point out it's a big grid: 8 rows across, 8 rows up.

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

The letters go **across** (a–h), the numbers go **up** (1–8). Every square has a name: letter first, then number. The bottom-left is **a1**.

**Quick game — "Find the square":** call out a square ("touch d4!", "touch h1!") and they point to it. Do 6–8. Make it fast and silly. Don't drill it — they'll absorb coordinates over weeks, not today.

---

## Part 2 — The rook moves in straight lines (5 min)

Put one rook (`R`) on **d4**. Ask: "Which way do you think it goes?" Then show them: the rook slides in **straight lines** — up, down, left, right — as far as it likes. The `*` squares are everywhere this rook can go:

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 | * | * | * | R | * | * | * | * |
  +---+---+---+---+---+---+---+---+
3 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

Say it like this: **"The rook makes a plus sign. Never diagonally."** Have them trace the lines with a finger.

---

## Part 3 — Blocking & capturing (5 min)

Two more rules:
1. **A rook can't jump over anyone.** If a friend is in the way, it has to stop before it.
2. **To capture,** it lands on an enemy's square and the enemy comes off the board.

Here `P` is a **friendly** pawn (blocks the rook) and `x` is an **enemy** the rook can capture. Notice the rook can go *up* only to d5 (friend on d6 blocks it), and *right* only as far as capturing on g4:

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   | P |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 | * | * | * | R | * | * | x |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   | * |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

**Ask them:** "Can the rook reach h4?" (No — the enemy on g4 blocks it, but it can *capture* that enemy.) "Can it reach d7?" (No — the friend on d6 is in the way.)

---

## Part 4 — Mini-game: "Rook Gobble" (8 min) 🍪

This is the main event. One rook, five pawns to eat. **No opponent** — it's a puzzle-game, and it builds the single most important beginner skill: *seeing the whole board.*

**Rules:** The rook (`R`) starts on e1. The pawns (`p`) don't move. Take turns making rook moves and capture pawns one at a time. Try to gobble **all five**. Then try again and beat your own move count.

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   | p |   |
  +---+---+---+---+---+---+---+---+
7 |   | p |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   | p |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   | p |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   |   |   |   | p |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   | R |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

**Coaching, gently:** if they reach for an illegal move (diagonal, or jumping), don't just say "no" — ask *"can the rook really get there in a straight line?"* Let them discover it. Praise the *thinking*, not just the right answer.

*(One clean solution exists in 6 moves: e1→e5 eats f5? no — talk it through together instead of chasing the record on day one. Finishing at all is a win.)*

---

## Part 5 — Mini-game: "Rook Race" (5 min) 🏁

A two-player finish. White rook `R` on a1, Black rook `r` on h1. **First rook to reach the far corner** (White to h8, Black to a8) wins. You alternate moves; you may **not** land on or pass through the other rook. Great for feeling how rooks move fast in straight lines.

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 | R |   |   |   |   |   |   | r |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

## Wrap-up (2 min)

Ask them to tell **you**: "How does a rook move?" (Straight lines, like a plus sign, can't jump.) That's the whole lesson — let them teach it back.

**Legend:** `R` = your rook · `r` = opponent rook · `p`/`P` = pawn · `x` = enemy you can capture · `*` = a square the rook can reach

**Stop while it's still fun.** If they want "one more Gobble," that's the perfect note to end on. Next time: the bishop (the diagonal slider) and another gobble game.
