# Chess — Day 4: The Knight

**For:** a 7-year-old beginner · **Time:** ~20–25 min, in short bursts · **You need:** a real board + pieces (or just this sheet)

**Today's rule of the day:** *The knight is the ONLY piece that can jump.* Two days of "no jumping!" — and today's piece breaks the rule. Kids love that. Fair warning: the knight's move is the hardest one in chess for everybody. Go slow, laugh a lot, and let fingers trace.

**Goal for today:** by the end, your child can make a legal knight move on purpose, and knows the knight leaps over pieces like they're not there.

---

## Part 1 — The L (5 min)

Put the knight (`N`) on **d4**. Teach the move as a walk: **"Two squares one way… then ONE square sideways."** Trace it with a finger every time, out loud: "one-two… turn!" The `*` squares are all eight landing spots:

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   | * |   | * |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   | * |   |   |   | * |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   | N |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   | * |   |   |   | * |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   | * |   | * |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

Two lovely noticings (don't force them, offer them): the landing spots make a circle around the knight — and every landing square is the **opposite color** from where he stands.

**Quick game — "Knight's telescope":** you point at a `*` square, they trace the L to it. Then swap: they point, you trace (get one wrong on purpose; let them catch you).

---

## Part 2 — The jump (5 min) 🎪

The showstopper. Fence the knight in with eight of your own pawns. A rook here would be stuck. A bishop would be stuck. The knight? Ask them: "Can he get out?"

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   | P | P | P | ★ |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   | P | N | P |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   | P | P | P |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

**One hop to f5, right over the fence.** Say it big: *"Knights don't care what's in the way. They FLY."* (Landing is different — he can't land ON a friend. Over is fine, on is not.)

---

## Part 3 — Mini-game: "Knight Gobble" (8 min) 🍪

Four enemy pawns to eat — and two of YOUR pawns parked right in the middle of the route. They don't block a knight. Every hop must gobble: **b1 takes c3, c3 takes d5, d5 takes f6, f6 takes h5.** Watch him sail over d4 and e5.

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   | p |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   | p | P |   |   | p |
  +---+---+---+---+---+---+---+---+
4 |   |   |   | P |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   | p |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   | N |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

**Coaching, gently:** knight moves go wrong in one of two ways — three-and-one ("too far!") or two-and-two ("that's a bishop step!"). When it happens, just re-chant the walk: *"one-two… turn."* Never let them feel dumb about it; adults blunder knight moves too.

---

## Part 4 — Knight's journey (4 min)

One quiet finisher on the real board: knight on **g1** (his real starting square in chess!), target **h5**, three hops, with your pawns on f2, g2, h2 exactly like a real game. First hop goes right over them. *(One route: g1→h3→f4→h5. In the app this is "knight-hop-03.")*

---

## Wrap-up (2 min)

Teach-back: "How does a knight move?" (An L: two, then one sideways.) "What makes him special?" (He's the only one who can JUMP.) If they're bouncing to show you the L with their whole body, today worked.

**Legend:** `N` = your knight · `p` = enemy pawn · `P` = your pawn · `*` = a landing square · `★` = the target square

**Stop while it's still fun.** Next time: the pawn — the littlest piece with the biggest dream. (The dream involves becoming a queen. Say no more.)
