# Chess — Day 3: The Queen

**For:** a 7-year-old beginner · **Time:** ~20 min, in short bursts · **You need:** a real board + pieces (or just this sheet)

**Today's rule of the day:** *The queen moves like a rook AND a bishop — but she's still just one piece, and she can't jump.* Nothing brand-new to learn today: it's the plus sign and the X glued together. That's on purpose — today is a power-up day, not a hard day.

**Goal for today:** by the end, your child can slide the queen along any of her eight roads on purpose (not by luck), and capture with her.

---

## Part 1 — Eight roads (4 min)

Put the queen (`Q`) on **d4**. Ask: "She moves like the rook AND the bishop. How many directions is that?" Count them together — eight:

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   | * |   |   |   | * |
  +---+---+---+---+---+---+---+---+
7 | * |   |   | * |   |   | * |   |
  +---+---+---+---+---+---+---+---+
6 |   | * |   | * |   | * |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   | * | * | * |   |   |   |
  +---+---+---+---+---+---+---+---+
4 | * | * | * | Q | * | * | * | * |
  +---+---+---+---+---+---+---+---+
3 |   |   | * | * | * |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   | * |   | * |   | * |   |   |
  +---+---+---+---+---+---+---+---+
1 | * |   |   | * |   |   | * |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

Say it like this: **"Plus sign PLUS the X — a big star."** Have them trace all eight roads with a finger. Then quiz, fast and silly: "Rook road or bishop road to h4?" (rook road) "To g7?" (bishop road).

---

## Part 2 — When the straight roads are blocked (5 min)

The queen's superpower is having a *second way* everywhere. Queen on **a1** wants **h7**. The up-the-side road is blocked by your pawn on... look — BOTH boring roads fail (the rank-7 road has a pawn on d7, the h-file road has a pawn on h4). Ask: "Rook roads are blocked. What would a bishop do?"

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   | P |   |   |   | ★ |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   |   |   |   | P |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 | Q |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

Two sparkling answers, both mixing a diagonal with a straight line: **a1→h8→h7**, or **a1→b1→h7**. Let them find either one. That mix — *diagonal plus straight* — is the whole lesson of Day 3.

## Part 3 — Mini-game: "Queen Gobble" (7 min) 🍪

Five pawns, scattered in every direction. The pawns don't move. Gobble **all five** — and notice how the queen uses rook roads for some and bishop roads for others.

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   | p |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   | p |   |   | Q |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   |   |   |   | p |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   | p |   |   |   | p |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

*(One clean tour, five moves, every move a gobble: e4 takes b7 on the diagonal, down the file to b4, diagonally to d2, along the rank to h2, and one small diagonal step to g3. There are other routes — theirs counts too.)*

**Coaching, gently:** if they get lost mid-slide, ask *"is that a rook road or a bishop road?"* Naming the road usually straightens it out.

---

## Part 4 — Efficiency round (4 min) ⚡

Four pawns, **four moves** — every move must gobble. This is Day 1's "one capture per move" game with eight directions instead of four. If a move eats nothing, start over and plan the chain first.

*(In the app this is "queen-gobble-03": pawns on a2, d2, c4 and f4, queen starting at c1.)*

---

## Wrap-up (2 min)

Teach-back time: "How does the queen move?" (Like a rook and a bishop — eight roads — but she can't jump.) Bonus question: "Who's stronger, the queen or the rook?" (The queen — and on Day 7 we'll find out *exactly* how much stronger.)

**Legend:** `Q` = your queen · `p` = enemy pawn · `P` = your pawn · `*` = a square the queen can reach · `★` = the target square

**Stop while it's still fun.** Next time: the knight — the ONLY piece in chess that can jump. After "rooks can't jump" and "bishops can't jump," that sentence lands like a firework.
