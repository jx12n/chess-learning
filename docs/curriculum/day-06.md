# Chess — Day 6: The King

**For:** a 7-year-old beginner · **Time:** ~15–20 min, in short bursts · **You need:** a real board + pieces (or just this sheet)

**Today's rule of the day:** *The king steps ONE square in any direction — and he is the most important piece on the board.* We are NOT learning check or checkmate today (that's the next adventure). Today the king just learns to walk, carefully, like the VIP he is.

**Goal for today:** by the end, your child can move the king exactly one square in any of the eight directions, and capture with him.

---

## Part 1 — One careful step (4 min)

Put the king (`K`) on **d4**. His whole move: one step, any direction. That's it. Eight neighbors, nothing more:

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   | * | * | * |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   | * | K | * |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   | * | * | * |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

Tell them why he's slow: **"He's the king. Kings don't run — everyone else runs FOR them. But if you lose him, you lose the game."** (That's all the check-talk we need today.)

**Quick game — "Royal walk":** call a nearby square ("walk to f6!") and have them count the steps. They'll discover the diagonal shortcut on their own: corner to corner of a 3×3 is 3 diagonal steps, not 6.

---

## Part 2 — Around the wall (4 min)

Kings never jump (of course — the dignity!). King on **d1** wants **d3**, but his own pawns wall the way. Let them find the walk *around*: **d1→c1→b2→c3→d3** (or the mirror road through e1 and f2). Four careful steps.

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   |   |   | ★ |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   | P | P | P |   |   |   |
  +---+---+---+---+---+---+---+---+
1 |   |   |   | K |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

## Part 3 — Mini-game: "King Gobble" (6 min) 🍪

The king captures the same way he walks: one step, landing on the enemy's square. Three pawns on a trail — march from snack to snack (a1 walks to b2, eats b3, strolls up, eats c5, eats d6):

```
  +---+---+---+---+---+---+---+---+
8 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
7 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
6 |   |   |   | p |   |   |   |   |
  +---+---+---+---+---+---+---+---+
5 |   |   | p |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
4 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
3 |   | p |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
2 |   |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
1 | K |   |   |   |   |   |   |   |
  +---+---+---+---+---+---+---+---+
   a   b   c   d   e   f   g   h
```

Then the wall-munch finisher: three pawns side by side right in front of him — **one bite per step, straight through the wall** (three moves, in the app it's "king-gobble-03").

**Coaching, gently:** the only king mistake is stepping too far. Ask *"how many squares does a king step?"* — "ONE" — and on they go. (When they one day learn about check, the careful habit will already be there.)

---

## Wrap-up (2 min)

Teach-back: "How does the king move?" (One square, any direction.) "Why do we keep him safe?" (Lose the king, lose the game.) Six pieces down — they now know how EVERY chess piece moves. Tell them that. It's a big deal.

**Legend:** `K` = your king · `p` = enemy pawn · `P` = your pawn · `*` = a square the king can reach · `★` = the target square

**Stop while it's still fun.** Tomorrow is the big finish of the first week: every piece together, and the secret of which prize to grab when you can only grab one.
